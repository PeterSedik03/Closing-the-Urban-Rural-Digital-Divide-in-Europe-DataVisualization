# Closing the Urban–Rural Digital Divide in Europe

This project was developed as part of a Data Visualization assignment during my MSc studies.  
The objective is to analyze territorial inequalities in household internet access across Europe, with a focus on the urban–rural digital divide.

## Problem
Despite the overall growth in internet access, significant territorial disparities persist across European countries, particularly between urban and rural areas.

## Data
The analysis is based on official Eurostat data:
- Households – level of internet access
- EU-level time series restricted to 2007–2019 to ensure consistency of the EU aggregate

A country-level comparison is conducted for 2024.

## Methodology
Data preparation and analysis were performed using:
- Microsoft Excel
- Power BI (Power Query)

The workflow included:
- data inspection and cleaning,
- reshaping from wide to long format,
- descriptive and comparative trend analysis.

No predictive or inferential models were applied; the focus was on transparent, interpretable comparisons.

## Results
The analysis shows:
- a strong increase in internet access across all territorial categories,
- faster growth in rural areas, although they still lag behind cities,
- towns and suburbs closely following urban trends.

This suggests that the digital divide in Europe is primarily a city–rural phenomenon.  
The 2024 country-level comparison highlights substantial heterogeneity across Europe, with some countries showing minimal gaps and others still facing significant rural disadvantages.

## Reproducibility and Data Literacy
The project explicitly addresses data limitations and ensures full reproducibility through a public GitHub repository containing:
- source datasets,
- Power BI files.
